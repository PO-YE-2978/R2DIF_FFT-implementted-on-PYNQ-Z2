## 1.1 為什麼要做這個專案？
* 本專案的目標，是從零開始設計一個可在 FPGA 上運行的 64-point FFT IP Core，並透過 PYNQ 平台，使用 Python 控制 FPGA 完成 FFT 運算。
* 實作內容不使用 FFT IP，而是自行設計 Verilog RTL 以實現 FFT 演算法與控制架構，因此能夠深入理解 FFT 硬體設計的每一個細節。
