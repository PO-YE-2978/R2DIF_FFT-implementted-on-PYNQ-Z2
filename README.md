# R2DIF_FFT-implementted-on-PYNQ-Z2
In this project, we designed and integrated a 64-point Radix-2 DIF FFT hardware using single Memory on a Xilinx Zynq SoC platform. Implemented the FFT core in Verilog, packaged it as an AXI4-Lite IP, integrated it into a Vivado Block Design, and controlled the accelerator through Python on the PYNQ platform for end-to-end hardware/software co-verification.
